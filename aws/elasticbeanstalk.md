# commands 
```
$ eb deploy
- デプロイコマンド

$ eb setenv key=value
- 設定の環境変数にkeyのvalueを追加

$ eb init 
- elasticbeanstalkアプリケーションを作成する

$ eb init --profile [profile_name]
- elasticbeanstalkアプリケーションを作成する（プロファイルを指定する）

$ eb ssh --setup
- elasticbeanstalk環境にSSH接続するための設定を行う
- キーを指定する

$ eb ssh
- elasticbeanstalk環境にSSH接続する
```
